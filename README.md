# 用户中心项目VUE版本

## 项目依赖安装

```shell
npm install
```

## 项目运行
```shell
npm run serve
```

## 项目打包
```shell
npm run build
```

# 关于本项目

声明：本项目仅供本人学习交流和使用，请勿盗用。

技术栈：

- 前端：vue3 + Antd-Vue版 + Axios + TypeScript + Vue Cil

后端暂时不介绍。

目前有的功能：管理员对用户增加和删除以及修改用户信息，用户注册，用户登录，用户个人中心(还未完善)。
